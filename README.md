## 一頁式摘要

- **市場成熟度**：高。NTT 2023 調查指出 69.7% 相關部門知道 social listening、58.5% 已實施、付費工具使用率 57.1%。日本不是教育市場，是替換/升級市場。
- **可公開驗證的工具市場規模**：日本 SNS マーケティング整體 2025 約 1 兆 1,171 億日圓，其中「分析工具」單獨估約 75 億日圓（約 1.4%）。Pandora 可服務市場保守抓 75 億日圓量級，含人工服務後實際 SAM 更大。
- **價格錨點**：
  - 入門：Buzz Finder 月額 88,000 日圓起
  - 中階：Buzz Finder Advanced 月額 198,000 日圓起、初期費用 110,000 日圓起
  - 風險監控含人工：月額 220,000 日圓起
  - 國際企業型（Meltwater / Brandwatch / Quid Monitor）：客製報價，常見 12 個月合約
- **競爭結構**：本地工具吃風險監控/低門檻；國際工具吃跨國品牌/深分析。中間帶（含 AI 洞察 + 行動建議）是 Pandora 切入機會。
- **產品策略**：以「監測 → 風險預警 → VOC 洞察 → 競品比較 → 自動報告 → 行動建議」一條龍包裝，定位為「AI ソーシャルインテリジェンス平台」，避免只做純 dashboard 比價。
