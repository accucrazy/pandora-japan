03Pandora 定價邏輯：關鍵字詞組 + 帳號

Pandora 的核心計價單位是「關鍵字詞組（キーワードグループ）」與「使用者帳號（ユーザー）」。這個模組化邏輯在日本剛好可以直接對齊 Buzz Finder、Quid Monitor 的計價方式，但更清楚可預測。

3.1 基本公式

月費 = 基礎平台費 (Platform Fee)
    + 關鍵字詞組數 × 詞組單價
    + 使用者帳號數 × 帳號單價
    + 選配模組（VOC / 競品比較 / API / BI / Managed）
＋ 初期費用 (Setup Fee, 一次性)

「關鍵字詞組」= 一個品牌 / 商品 / 競品 / 議題的監測單元，通常含主詞、同義詞、typo、排除字、布林邏輯的一整組 query。客戶的感知是「每多一個品牌 / 商品就多一組」，非常直覺好預算化。

為什麼這個計價結構適合日本市場
  • 日本採購方習慣「逐項報價 / 見積書」：詞組、帳號、模組拆開明列，報價透明、利於過內部稟議。
  • 對標 Buzz Finder 的「轉送數量」計價，Pandora 的「詞組」更貼近客戶業務語言（品牌數＝詞組數）。
  • 帳號分層定價讓 Plan 2 → Plan 3 自然升級：人數一多，Plan 3 的 2,500 円/名比 Plan 2 的 3,500 円/名更划算。
  • 初期費用（Setup Fee）是日本 SaaS 的常見錨點，利於前期 Onboarding 收費 cover 成本。

對標市場的 3 個核心打法
  1. 用「公式」打「黑箱」：Quid / Meltwater / Brandwatch 的共同弱點是「見積不出來、採購稟議通不過」。Pandora 把詞組、帳號、模組全部明碼，日本客戶拿到報價單就能直接跑決裁 flow。
  2. 用「詞組」打「轉送」：Buzz Finder 計價邏輯是「轉送量」，客戶很難預測月底帳單。Pandora 的「品牌數 = 詞組數」是可控的採購單位，IT / 行銷都看得懂。
  3. 用「模組內含」打「加購」：Meltwater GenAI Lens、Quid Generative 都是另外加購，Pandora 在 Plan 2 原生內含 AI 問答 + VOC 洞察，同價不同值。

  • 話術：「同價格、自動做完月報」
  • 通路：與公關公司合作替換
vs Social Insight（運營型）
  • 明確區隔：不做發文、只做洞察
  • P2 補足 VOC / 競品深度
  • 話術：「運營留 Social Insight、分析換 Pandora」
  • 可共存，甚至透過 API 餵分析
vs Quid Monitor（深分析）
  • P3 起價 60 萬円 vs Quid 100 萬円
  • 差異：UI 更友善、AI 內含不加購
  • 話術：「深度 80%，價格 50%」
  • 攻點：中型品牌 Quid 用不完的場景
vs Meltwater / Brandwatch（國際）
  • 不跟 PR Wire 比、聚焦 social
  • 強化日文 NLP + 5ch / Review 在地平台
  • 年約 300 萬円起 vs 對手 600 萬円+
  • 話術：「日本語精度 + 合約彈性」

05潛在成本（重心：一年內鋪滿全渠道爬蟲）

成本模型把絕大多數預算投在 Y1 全渠道爬蟲一次鋪完，其他（銷售、行銷、合規）維持最精簡。Y2、Y3 以 維運 + 擴展 為主。全部以新台幣（TWD）標示，匯率採 1 TWD ≒ 4.7 JPY。

Y1 分兩階段：先日文版 MVP、再擴日本在地渠道
  • 階段 1｜日文版（Y1 Q1–Q2，約 3 個月）：日文 UI 全面本地化、上 X（雙軌）+ 基礎日本新聞 RSS，即可進入市場賣 Plan 1 風險監測。不動日本在地渠道。
  • 階段 2｜日本在地渠道（Y1 Q3 起，約 6 個月）：再投入 1 名 RD × 6 個月建 Threads / Instagram / 5ch / Review / 新聞深化，解鎖 Plan 2 VOC + 競品洞察。
  • YouTube / TikTok 延至 Y2 視客戶需求再啟動。
核心決策：X 採「X API + 自寫爬蟲」雙軌評估
  • A 軌｜X API（Pro / Enterprise tier）：合法、穩定、即時；缺點是授權費高、條款隨 X 公司政策變動。
  • B 軌｜自寫爬蟲：成本可控、規格可自訂；缺點是須對抗反爬 / 帳號風控、法務風險較高。
  • Y1 前 6 個月雙軌並行，比較「資料完整度、延遲、單則成本、法務風險」，Y1 H2 決定主軌（或維持雙軌 fallback）。Y1 預算：API 約 NT$2.5M、自寫爬蟲 dev 約 NT$1.5M（含 1 名工程師 6 個月、代理 IP、反爬工具、法務諮詢）。
