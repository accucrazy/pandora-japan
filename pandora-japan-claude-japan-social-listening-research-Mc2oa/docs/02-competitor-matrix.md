## 3. 對手定位四象限（質性判斷）

```
                  深分析能力 高
                       ▲
                       │
       Quid Monitor JP │ Brandwatch
                       │ Meltwater
                       │
    Buzz Finder ───────┼───────────────► 全球品牌覆蓋廣
                       │
       Social Insight  │
                       │
                       ▼
                  深分析能力 低
```

- 左下：本地、運營型、價格透明
- 左上：本地、企業深分析
- 右上：全球、深分析、跨市場
- 右下：偏 SNS 運營一體解

**Pandora 應切入的機會空缺**：左上偏中段（日本企業中型客戶 / 在地大型品牌的「AI 洞察 + 行動建議」帶），用 Buzz Finder 級別的入門價錨 + Quid / Meltwater 級別的洞察價值打中間市場。

## 4. 各家差異化要點摘要

### Buzz Finder
- 賣點：日系信任 + 透明月費 + 24x365 人工 + X 全量
- 弱點：洞察深度與多語非主力
- Pandora 對策：在「分析深度」與「AI 報告自動化」上拉開差距

### Social Insight
- 賣點：SNS 運營與分析一體、所有功能無額外 option fee
- 弱點：偏帳號與運營，研究型分析較弱
- Pandora 對策：明確區分「我們不做運營，做洞察與決策」

### Quid Monitor Japan
- 賣點：多來源 / 多語 / BI / 生成 AI / Data Uploader
- 弱點：價格客製、入門門檻高、UI 偏研究型
- Pandora 對策：用更友善的 UI、清楚的入門包 land 中型客戶

### Meltwater
- 賣點：PR + 媒體 + 社群 + Influencer 一體
- 弱點：契約期長（多 12 個月）、價格不透明
- Pandora 對策：合約彈性 + 透明定價 + 在地化導入支援

### Brandwatch
- 賣點：全球研究等級、數十億對話資料
- 弱點：日本在地化深度不如本地廠
- Pandora 對策：強化日本語 NLP、在地平台 coverage、本地客戶成功
