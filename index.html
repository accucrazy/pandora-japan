  • 入門方案 88k 円月約進場
  • 差異：AI 月報 / 情緒細分 / 主題萃取自動化
  • 通路：與公關公司合作做替換
  • 話術：「同價格、自動做完月報」

  • 日文 NLP + 5ch / Review 差異化
  • 年約 300 萬円起 vs 對手 600 萬円+
  • 話術：「日本語精度 + 合約彈性」
三個共通打法
  • 用「公式」打「黑箱」：Pandora 明碼報價直接過稟議
  • 用「詞組」打「轉送量」：可控採購單位
  • 用「Datalake」打「Dashboard」：資料可二次分析

04Pandora 日本方案規格

方案分 入門 / 標準 / 進階 三階，差異只在兩軸：① 運算資源（爬蟲速度）、② Datalake 容量（Token 額度）。三個方案 登入帳號統一 5 組，超過 NT$25,000 / 月 / 帳號（約 ¥117,500 / 月）。

4.1 產品兩大差異化：Token + Datalake

① Token（LLM 算力單位）
  • 摘要、VOC 萃取、競品比較、AI 問答、輿情監測報告生成，全部消耗 LLM Token
  • Token 是客戶可感知的「分析預算」單位，用多少 = 產出多少報告
  • 每方案內含 Datalake 月 Token 額度，超額按量計價（輸入 ¥0.5 / 輸出 ¥1.5 每千 tokens）
  • 客戶好處：預算可控、按量擴充
② Datalake（原生資料資產）
  • 所有 X / Threads / IG / 5ch / Review / News 資料進 GCP BigQuery Datalake（東京 Region，APPI 合規）
  • 兩個容量等級：Data Lake 50M tokens / Data Ocean 150M tokens
  • 客戶可 ad-hoc query、歷史回溯、跨渠道關聯（非 pre-aggregated dashboard）
  • 進階方案可 BI / API 直連資料湖，自家 BI / CDP / LLM 二次分析
  • 競品比較：Buzz Finder / Social Insight 只給 UI；Meltwater 資料需額外購買 export 權

4.3 Datalake 容量 ↔ 輿情監測報告產出換算

以 Data Ocean 150M Tokens 為基準（約 900 份報告／月），依品牌聲量高低換算如下：

品牌聲量 單份報告文章數 Data Ocean（150M） Data Lake（50M） 典型應用
低聲量 30 篇 / 份 約 1,000 份 約 333 份 利基品牌、區域商家、B2B
混合（基準） — 約 900 份 約 300 份 一般品牌組合監測
高聲量 500 篇 / 份 約 125 份 約 42 份 頭部品牌、熱門議題、危機監測

※ 一份報告 = 抓取相關文章 → LLM 摘要 + 情緒分析 + 主題萃取 + 結論建議，Token 消耗隨文章數線性成長。

為什麼這個結構適合日本
  • 只差兩軸、決策簡單：客戶只要回答「品牌聲量大小」＋「要不要即時」就能選對方案，過稟議書極快。
  • 報告產出量可預測：Token ↔ 報告份數的換算表讓採購方一目了然，避開競品「無限流量但實際卡頓」的陷阱。
  • Datalake 帶來資料資產感：GCP 東京 Region + 合約期資料保留，解日本大企業最在意的「資料留在自己這」。
  • 帳號統一 5 組：大多數日本行銷團隊規模 3–5 人，預設即夠；如跨部門共用，NT$25k / 月 / 人彈性擴充。

05研發開發成本

日本市場進入分為 3 個 Phase，總計 12 人月、一次性投入 NT$120 萬，類似製造業的「開模費」。此研發費用由騰雲（母公司）支付，Pandora Japan 不承擔此一次性成本。

5.1 開模費：NT$120 萬（三階段、12 人月，由騰雲支付）

階段內容里程碑工期人月小計

※ 人月單價 NT$10 萬 / 月，2 名工程師並行可壓縮至 6 個月日曆月內完成全部三階段（Phase 1 完成後即可開始 Phase 2 / 3 並行）。

「開模費」的意義（製造業類比）
  • 類似射出成型開模：NT$1.2M 是一次性 tooling 費用，由騰雲（母公司）支付，做完後日本產線可持續複用。
  • Pandora Japan 拿到的是「已開模的產品」：不用自己付研發，直接進市場銷售。
  • 騰雲的回本邏輯：NT$10 萬 / 月 × 12 個月年約 = NT$1.2M → 1 家標準 / 進階年約客戶就抵掉整筆開模費。
  • 後續所有新簽的日本客戶，對騰雲都是淨 ROI 貢獻。

5.3 Token 與 Datalake 的成本結構

成本項採購成本計入 Pandora 客戶方案的方式
AI / LLM Token 向 OpenAI / Anthropic 採購：輸入 $0.003 / 千、輸出 $0.015 / 千（GPT-4 級） Plan 內含月額度，超額 ¥0.5–1.5 / 千 tokens（毛利 ~60%）
BigQuery（Datalake 查詢） ~$5 / TB 掃描、儲存 $0.02 / GB-月 入門 Data Lake 50M / 標準・進階 Data Ocean 150M；超額延伸付儲存費
Pub/Sub（即時告警） ~$0.04 / 百萬訊息 入門日報；標準 / 進階即時告警內含
GKE（爬蟲 / API 服務） 依 CPU / Memory 使用量 共用叢集，固定成本攤提

※ Token 是變動成本（隨客戶使用量變化）、Datalake 是固定 + 少量變動成本；兩者毛利率分別 ~60% 與 ~75%，整體 COGS 毛利率維持 70%+。
