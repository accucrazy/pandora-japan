# 01. 日本社群輿情市場現況

## 1. 市場定義

本研究涵蓋的「Pandora 同類型產品」在日本被稱為：

- ソーシャルリスニング（Social Listening）
- 口コミ分析 / VOC 分析
- ソーシャルメディア分析
- コンシューマーインテリジェンス（Consumer Intelligence）
- 一部歸類於 PR / メディアモニタリング

對應的核心使用者部門：マーケティング、ブランド、広報・PR、リスク管理、商品企劃、CX、研究開發。

## 2. 主要使用情境

依日本市場已被教育好的需求，大致 5 類：

1. 炎上 / 風評 / 事故前兆監測（風險管理）
2. 品牌與競品聲量比較（行銷）
3. 商品 / 活動 / 代言反應觀測（行銷、商品）
4. VOC 痛點 / 喜好萃取（商品、CX）
5. 多市場 / 多語言 intelligence（跨國品牌、海外事業）

NTT 2023 調查指出，企業使用 social listening 的最主要目的就是「盡早掌握炎上與 trouble」，其次才是行銷分析。

## 3. 主要對標廠商

### 3.1 日本本地

| 廠商 | 產品 | 切入點 |
| --- | --- | --- |
| NTT ドコモビジネス X | Buzz Finder | X 全量、即時告警、炎上監測、低門檻月費 |
| User Local | Social Insight | SNS 運營 + 口碑分析整合 |
| TDSE | Quid Monitor Japan（前 NetBase） | 多來源、多語言、深度分析、BI 串接、研究型 |

### 3.2 國際企業型

| 廠商 | 產品 | 切入點 |
| --- | --- | --- |
| Meltwater | Social Listening / Consumer Intelligence / Media Intelligence | PR + 媒體 + 社群一體 |
| Brandwatch | Consumer Intelligence / Consumer Research | 全球級研究型、跨國品牌 |

> 在日本市場 Brandwatch / Meltwater 多透過在地團隊或合作夥伴推進；公開價格表多為「客製報價」。

## 4. 規格現況（市場已被教育好的標配）

整理自上述 5 家公開頁面，日本企業客戶對此類產品的期待規格大致為：

### 4.1 資料來源
- X / Instagram / Facebook / YouTube / TikTok / Threads
- ブログ / 掲示板 / ニュース / レビューサイト
- X 全量資料可回溯多年（Quid Monitor 標榜可回溯到 2006 年）
- 多語言（Quid Monitor 約 50 種語言）

### 4.2 監測與告警
- 即時或近即時收集
- 異常聲量 / 負面比例急升告警
- 炎上前兆偵測
- 24x365 人工監控與風險判定（Buzz Finder 等本地服務的賣點）

### 4.3 分析功能
- Sentiment（正/中/負，並逐步擴及攻擊型、抱怨型等）
- SOV / 競合比較
- 受眾屬性分析、地域分析
- 趨勢偵測
- VOC 萃取（pain / need / praise）
- 自有資料（CRM / 問卷 / Review）混合分析

### 4.4 操作與整合
- 中文/日文 UI 易用性（NTT 調查顯示 UI 為「最被重視」與「最常被抱怨」項目同時上榜）
- 噪音 / spam 去除（同調查指出滿意度最低項目）
- 報表自動化、PDF / 圖檔匯出
- BI 串接（Tableau / Power BI / Domo / Datorama）、API、Webhook

### 4.5 AI 化趨勢
- 生成 AI 問答式分析（Quid 已主打）
- AI 摘要報告（Meltwater 等已將 GenAI Lens 進入定價結構）

## 5. 計價方式

日本市場存在三種典型計價模式：

### A. 入門型月費
- 代表：Buzz Finder
- 月額 88,000 日圓起，含基本監測 / 日報 / 告警

### B. 中高階月費 + 初期費用
- 代表：Buzz Finder Advanced
- 初期費用 110,000 日圓起、月額 198,000 日圓起
- Buzz Finder 風險監測 + 人工告警方案：月額 220,000 日圓起

### C. Enterprise 客製報價
- 代表：Quid Monitor Japan / Meltwater / Brandwatch / Social Insight
- 通常依以下維度報價：
  - 監測來源與地域
  - 歷史資料深度
  - 關鍵字 / Query 數量
  - 使用者帳號數
  - API / BI 串接需求
  - Onboarding / Customer Success / Analyst Support
  - Managed Service（24x365 監控、危機應變、代寫報表）
- Meltwater 官方明示 12 個月合約最常見

## 6. 市場規模推估

| 項目 | 數值 | 來源／註解 |
| --- | --- | --- |
| 日本 SNS マーケティング整體市場規模（2025 預估） | 約 1 兆 1,171 億日圓 | 含廣告、網紅、代運營、分析工具、企劃顧問 |
| 其中「分析工具」 | 約 75 億日圓（約 1.4%） | 最接近 Pandora 類產品的可比 benchmark |
| 知道 social listening 的相關部門比例 | 69.7% | NTT 2023 調查 |
| 已實施比例 | 58.5%（在知道者中） | 同上 |
| 近一年內實施 | 52.3%（在已實施者中） | 同上 |
| 付費工具使用比例 | 57.1%（在已實施者中） | 同上 |

**Pandora 在日本可服務市場（SAM）保守判斷**：
- 純工具 SaaS：約 75 億日圓量級
- 含 Managed Service / 顧問報表 / 危機應變代運營 / 海外監測：實際更大，但目前無公開可乾淨驗證的數字

## 7. 龍頭與市占判斷（限制聲明）

> 目前**沒有**找到日本公開、可驗證的 vendor market share 排名。下表為依公開能見度、在地化程度、導入語境的方向性分組，不是實際市占數字。

- **本地強勢**：Buzz Finder（NTT）、Social Insight（User Local）、Quid Monitor Japan（TDSE）
- **國際強勢**：Meltwater、Brandwatch
- **本地與國際的差異**：本地工具吃風險監控、日系客戶、低門檻導入；國際工具吃大型企業、跨國品牌、整合 intelligence。

## 8. 對 Pandora 的市場啟示

1. 日本不是教育市場，是替換／升級市場。產品說明可直接跳過「為什麼要 social listening」。
2. 「風險監控」是最容易被預算批准的入口，建議作為 land 動作。
3. 真正的差異化在「從發現到行動」，不只是 dashboard。
4. 日本語 NLP 精度與在地平台 coverage（X / Threads / 5ch / Review / News）會直接決定成交。
5. 定價必須有清楚的 land-and-expand 三層結構，全客製反而難切入中型客戶。
