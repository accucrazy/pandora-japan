# 04. 進場策略與定價建議

## 1. 定位 Statement

> Pandora Japan 是面向日本市場的 AI ソーシャルインテリジェンス平台，協助品牌從輿情監測到風險預警、VOC 洞察、競品分析與行動建議，一條龍完成。

## 2. 定價帶建議

依日本市場已存在的價格錨點對齊：

| 方案 | 月額（建議） | 初期費用 | 對標 | 主要功能 |
| --- | --- | --- | --- | --- |
| Plan 1 – Monitor | 88,000 ～ 120,000 日圓 | 0 ～ 50,000 日圓 | Buzz Finder 入門 | 3-5 組關鍵字、即時告警、日報、基礎 dashboard、1-3 帳號、Email/Chat 通知 |
| Plan 2 – Insight | 200,000 ～ 350,000 日圓 | 100,000 日圓 | Buzz Finder Advanced + Quid 入門帶 | 10+ 關鍵字、競品比較、VOC 洞察、月報、Campaign 分析、簡報素材匯出、5+ 帳號 |
| Plan 3 – Enterprise | 600,000 日圓起，客製 | 300,000 日圓起 | Quid Monitor / Meltwater | 多品牌 / 多市場、Data Uploader、API/BI、多語、SSO、顧問、24x365 Managed |

**定價邏輯**：
- 入門卡在 Buzz Finder 同價帶，避免被當「貴一截」的選項。
- 中階刻意比 Buzz Finder Advanced 同價或略高 10%，但價值差異打在「AI 洞察 + 自動報告」。
- Enterprise 走客製，但給「具體起價」做信任錨點，避免全黑箱。

## 3. 進場策略

### 3.1 Land：用 Plan 1 切入風險監測場景
- 預算來源：公關 / 風險管理（不需經行銷部 RFP）
- Sales pitch：「お試しで一ブランド・一ヶ月から導入可能」
- KPI：3 個月內 30 ～ 50 家中型 logo

### 3.2 Expand：升級到 Plan 2（VOC + 競品）
- 觸發點：Plan 1 客戶第一次想做季度 review、活動檢討
- 升級流程：自動推送「Insight 試用報告」（用客戶現有資料生成 1 份 demo 月報）
- KPI：Plan 1 → Plan 2 升級率 30%+

### 3.3 Enterprise：跨品牌 / 跨市場
- 觸發點：Plan 2 客戶已有 5+ 帳號、需要 BI 串接、需要海外監測
- 銷售方式：1-1 顧問式、走 6-12 個月合約
- KPI：Plan 2 → Plan 3 升級率 10%+，ACV 6 倍以上

## 4. 通路策略

| 通路 | 目的 | 備註 |
| --- | --- | --- |
| 直銷 | Plan 2 / Plan 3 主力 | 配置在地 BD + Customer Success |
| 廣告代理商合作 | Plan 1 / Plan 2 量產 | 與既有電通系、博報堂系、獨立代理商共售 |
| PR 顧問公司合作 | Plan 1 風險監測導入 | 借助公關公司既有客戶關係 |
| BI / SI 廠商 | Plan 3 串接型導入 | 對接 Tableau / Power BI / Domo 既有導入專案 |

## 5. 在地化必做項

| 類別 | 內容 |
| --- | --- |
| 語言 | UI / 文件 / 客服 / 銷售素材全日文 |
| 法規 | 個人情報保護法（APPI）合規、資料儲存地宣告 |
| 安全 | ISMS（ISO/IEC 27001）認證計畫；Enterprise 客戶會問 |
| 服務 | 日本時區客服窗口、緊急聯絡 SOP |
| 計費 | 日圓報價、發行請求書（インボイス制度合規） |
| 合約 | 日文版 MSA、NDA、DPA |

## 6. 關鍵成交障礙與對策

| 障礙 | 對策 |
| --- | --- |
| 「為什麼要換掉現有工具」 | 提供「現工具 vs Pandora」並排報告，免費 1 個月 PoC |
| 「日本語精度可信嗎」 | 提供日本語 NLP benchmark 結果 + 客戶 logo 引用 |
| 「資料安全與合規」 | ISMS 認證、APPI 對應頁、資料儲存說明 |
| 「契約期太長」 | 提供 1 個月 / 3 個月短約彈性（Plan 1） |
| 「沒有在地團隊」 | 投入日本 BD + CS，至少 1 名日語母語 PM |

## 7. 上市時程建議

| 時段 | 里程碑 |
| --- | --- |
| M0-M2 | P0 規格凍結、日文 UI 與在地素材完成、ISMS 計畫啟動 |
| M3-M4 | 5-10 家在地客戶 PoC（含一家公關公司、一家零售品牌、一家跨國品牌日本分公司） |
| M5-M6 | Plan 1 / Plan 2 GA，公開定價發布 |
| M7-M9 | Plan 3 客製案例落地（含 BI 串接 / Data Uploader） |
| M10-M12 | Managed Service 上線，目標 ARR 進入下一階段 |

## 8. 成功指標

- **產品**：Day-7 Activation ≥ 60%、月報自動生成使用率 ≥ 70%、告警準確率 ≥ 85%
- **商務**：Plan 1 月新增 logo ≥ 10、Plan 1 → 2 升級率 ≥ 30%、年解約率 ≤ 15%
- **品牌**：在「ソーシャルリスニング 比較」搜尋結果第一頁出現、3 家以上日經登載品牌 logo

## 9. 風險與應對

| 風險 | 應對 |
| --- | --- |
| Buzz Finder 降價反擊 | 強化 AI 洞察與自動報告，避開純價格戰 |
| Quid Monitor / Meltwater 在地化加速 | 加速 Plan 2 在地客戶 logo 與案例 |
| 日本市場保守，PoC 拉長 | 提供「3 週見成果」標準化 PoC 流程 |
| 日本語 NLP 精度落差 | 與在地 NLP 團隊或學界合作建立日本語語料 |
| 個資 / 安全事件 | 走 ISMS、第三方滲透測試、明確 incident response SOP |
